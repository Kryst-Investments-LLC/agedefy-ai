# Support

| Channel | Use it for |
|---|---|
| GitHub Issues | Bug reports, feature requests, schema gaps |
| GitHub Discussions | Architecture questions, RFCs |
| security@agedefy.ai | Security reports (see [SECURITY.md](SECURITY.md)) |
| conduct@agedefy.ai | Code-of-conduct reports |
| legal@agedefy.ai | Jurisdiction / rule corrections |

Please **never** include real user PHI, biomarker data, API keys, or
customer-identifying information in any public channel.

## Response targets

| Severity | First response | Resolution target |
|---|---|---|
| Critical (security, data loss, broken legal block) | 1 business day | 5 business days |
| High (workflow regression, CI failure) | 2 business days | 10 business days |
| Normal | 5 business days | best-effort |
| Low / docs | best-effort | best-effort |
